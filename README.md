# README

## Зависимости

- Ruby 3.1.2
- Node.js 16.17.1

## Для разварота проекта

1. Сколнируйте проект
2. Установите зависимости
3. Сделайте `bundle install`
4. Сделайте `bundle exec rails db:migrate`
5. Следайте `bundel exec rails s`

Должно получитсься)))
